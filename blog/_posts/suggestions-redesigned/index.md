As you know, on the homepage or via the [browser extension](/browser-extension/), you can type in a letter or word and see a list of matching shortcuts.

I’ve redesigned these suggestions:

{% imageFull "img/gfl.jpg", "Suggestions for 'gfl'" %}

You can dive deeper by clicking:

- the query example(s) to see exactly where that command redirects, and
- a tag (yellow) to explore related shortcuts.

The buttons in the footer point to the GitHub repo, where you can directly report issues or send pull requests.
