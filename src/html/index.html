Using Advanced settings via Github. Basic settings disabled.

Set advanced settings and personal shortcuts with a (free) Github account.

Example queries
g berlin
search Google for "berlin"
w berlin
open Wikipedia article for Berlin (in language from settings)
fr.w berlin
override language from settings: use French Wikipedia
gd london, manchester
find a route on Google Maps from London to Manchester
gfl ber, lpa
Google Flights search from Berlin to Las Palmas
ytd cats
Cat videos on Youtube, sorted by date
About
  • privacy: all queries are processed in the client, not server
  • social: benefit from curated shortcuts in the general repo, organized by namespaces.
  • freedom: setup your own personal shortcuts in your own repo
