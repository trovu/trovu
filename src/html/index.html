Examples

          

            

              gfl ber, nyc
            

            
Search on Google Flights for a flight from Berlin to New York City

            

              gfl ber, nyc, fr, 28
            

            
… leaving next Friday, returning on the 28th

            

              gd london, liverpool
            

            
Search for a route on Google Directions from London to Liverpool

            

              g berlin
            

            
Search Google for Berlin

            

              w berlin
            

            
Go to the Wikipedia article about Berlin

            

              fr.w berlin
            

            
Go to the French Wikipedia article about Berlin

            

              wg berlin
            

            
Search Wikipedia for all mentions of "berlin" via Google

            

              npm csv
            

            
Search the Node Package Manager for modules projects about CSV
